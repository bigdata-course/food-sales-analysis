## food-sales-analysis-visualization

本工程是展示数据分析与预测结果的可视化平台，是一个简易的网站。

## Software Development Environment
--Java JDK1.8.0_101  
--apache-tomcat-8.5.6  
--SQL Server 2008 R2  
--MyEclipse 2014  
--springMVC  
--hibernate
